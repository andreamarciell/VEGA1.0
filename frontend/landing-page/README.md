# Toppery Landing Page

This is the standalone landing page for Toppery, designed to be deployed separately on Netlify.

## Features

- Modern, responsive landing page
- Product showcase
- Technology stack presentation
- Testimonials section
- Call-to-action sections
- Contact information

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for Netlify deployment. Simply connect your repository to Netlify and it will automatically build and deploy.

## Structure

- `src/components/landing/` - Landing page specific components
- `src/components/ui/` - Reusable UI components
- `src/content/` - Content and translations
- `src/lib/` - Utility functions and configurations
